## What's Coming

The WebSocket API will provide real-time streaming for:

- **Price Updates** - Live price changes as they happen
- **Orderbook Streams** - Real-time orderbook depth updates
- **Trade Feed** - Stream of executed trades
- **Market Events** - Resolution announcements and status changes

## Example Usage

```javascript
// Coming soon
const ws = new WebSocket('wss://ws.predexon.com');

ws.onopen = () => {
  ws.send(JSON.stringify({
    action: 'subscribe',
    channel: 'prices',
    markets: ['will-trump-win-2024']
  }));
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Price update:', data);
};
```
