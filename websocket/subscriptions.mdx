## Subscribe

Create a subscription to receive real-time alerts matching your filters.

### Request

```json
{
  "action": "subscribe",
  "platform": "polymarket",
  "version": 1,
  "type": "orders|activity|lifecycle",
  "filters": {
    "users": ["0x..."],
    "condition_ids": ["0x..."],
    "market_slugs": ["slug"]
  }
}
```

### Fields

| Field | Required | Description |
|-------|----------|-------------|
| `action` | Yes | `"subscribe"` |
| `platform` | Yes | `"polymarket"` |
| `version` | Yes | `1` |
| `type` | Yes | Channel: `"orders"`, `"activity"`, or `"lifecycle"` |
| `filters` | Yes | At least one filter type required |

### Filter Types

| Filter | Description | Example |
|--------|-------------|---------|
| `users` | Wallet addresses | `["0x123...", "0xabc..."]` |
| `condition_ids` | Market condition IDs | `["0x456...", "0x789..."]` |
| `market_slugs` | Market URL slugs | `["will-donald-trump-win-the-2024-us-presidential-election"]` |

### Success Response

```json
{
  "type": "ack",
  "subscription_id": "sub_a1b2c3d4",
  "channel": "trades"
}
```

Save the `subscription_id` to update or unsubscribe later.

## Examples

### Subscribe to trade alerts for wallets

Track trades from specific wallet addresses:

```json
{
  "action": "subscribe",
  "platform": "polymarket",
  "version": 1,
  "type": "orders",
  "filters": {
    "users": [
      "0x1234567890abcdef1234567890abcdef12345678",
      "0xabcdefabcdefabcdefabcdefabcdefabcdefabcd"
    ]
  }
}
```

### Subscribe to activity for a market

Track splits, merges, and redemptions on a specific market:

```json
{
  "action": "subscribe",
  "platform": "polymarket",
  "version": 1,
  "type": "activity",
  "filters": {
    "condition_ids": ["0x1234567890abcdef1234567890abcdef12345678901234567890abcdef12345678"]
  }
}
```

### Subscribe to lifecycle events by slug

Track token registrations and market resolutions:

```json
{
  "action": "subscribe",
  "platform": "polymarket",
  "version": 1,
  "type": "lifecycle",
  "filters": {
    "market_slugs": ["will-donald-trump-win-the-2024-us-presidential-election"]
  }
}
```

### Wildcard subscription (Pro/Desk only)

Subscribe to **all** trades across Polymarket:

```json
{
  "action": "subscribe",
  "platform": "polymarket",
  "version": 1,
  "type": "orders",
  "filters": {
    "users": ["*"]
  }
}
```

<Warning>
**Pro:** Wildcard must be the only subscription on the connection. One wildcard connection per API key.

**Desk:** Wildcard can be mixed with other subscriptions. Up to 10 wildcard connections per API key.
</Warning>

## Unsubscribe

Remove an existing subscription using its ID.

### Request

```json
{
  "action": "unsubscribe",
  "subscription_id": "sub_a1b2c3d4"
}
```

### Response

```json
{
  "type": "ack",
  "subscription_id": "sub_a1b2c3d4"
}
```

---

## Update Subscription

Modify filters on an existing subscription without creating a new one.

### Request

```json
{
  "action": "update",
  "subscription_id": "sub_a1b2c3d4",
  "filters": {
    "users": ["0xnewwallet..."]
  }
}
```

### Response

```json
{
  "type": "ack",
  "subscription_id": "sub_a1b2c3d4"
}
```

<Note>
Updating replaces the existing filters entirely. Include all items you want to track.
</Note>
