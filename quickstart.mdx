## Overview

The Predexon API provides RESTful endpoints for accessing prediction market data. This guide will walk you through making your first API calls.

<Steps>
  <Step title="Base URL">
    All API requests should be made to:
    ```
    https://predexon-data-service-8970529761.us-central1.run.app
    ```
  </Step>
  <Step title="Make Your First Request">
    Let's fetch the top markets by trading volume:

    <CodeGroup>
    ```bash cURL
    curl "https://predexon-data-service-8970529761.us-central1.run.app/v1/polymarket/markets?status=open&sort=volume&limit=5"
    ```

    ```python Python
    import requests

    BASE_URL = "https://predexon-data-service-8970529761.us-central1.run.app"

    response = requests.get(
        f"{BASE_URL}/v1/polymarket/markets",
        params={
            "status": "open",
            "sort": "volume",
            "limit": 5
        }
    )

    data = response.json()
    for market in data["markets"]:
        print(f"{market['title']}: ${market['total_volume_usd']:,.0f} volume")
    ```

    ```typescript TypeScript
    const BASE_URL = "https://predexon-data-service-8970529761.us-central1.run.app";

    const response = await fetch(
      `${BASE_URL}/v1/polymarket/markets?status=open&sort=volume&limit=5`
    );

    const { markets } = await response.json();
    markets.forEach(market => {
      console.log(`${market.title}: $${market.total_volume_usd.toLocaleString()} volume`);
    });
    ```
    </CodeGroup>
  </Step>
  <Step title="Explore the Response">


  </Step>
</Steps>

## Common Use Cases

### Get Market Price History

Fetch candlestick data for charting:

```python
response = requests.get(
    f"{BASE_URL}/v1/polymarket/candlesticks/{condition_id}",
    params={
        "interval": 60,  # 1-hour candles
        "start_time": 1704067200,
        "end_time": 1704153600
    }
)
```

### Track a Wallet's Positions

Get all open positions for a wallet address:

```python
response = requests.get(
    f"{BASE_URL}/v1/polymarket/wallet/positions/{wallet_address}",
    params={"sort_by": "value", "limit": 50}
)
```

### Find Cross-Platform Matches

Discover equivalent markets on Kalshi:

```python
response = requests.get(
    f"{BASE_URL}/v1/matching-markets",
    params={"polymarket_market_slug": "will-trump-win-2024"}
)
```

The API currently has generous rate limits suitable for most use cases:

| Tier | Requests/minute | Requests/day |
|------|-----------------|--------------|
| Free | 60 | 10,000 |
| Pro | 300 | 100,000 |

<Note>
Contact us if you need higher limits for production applications.
</Note>

## Error Handling

The API uses standard HTTP status codes:

| Code | Description |
|------|-------------|
| `200` | Success |
| `400` | Bad request - check your parameters |
| `404` | Resource not found |
| `429` | Rate limit exceeded |
| `500` | Internal server error |

Error responses include helpful messages:

```json
{
  "error": "Invalid filter combination",
  "message": "Only one of market_slug, token_id, or condition_id can be provided"
}
```
