## Overview

Returns deposit addresses for funding a Polymarket wallet from other chains. Supports deposits from Ethereum, Solana, Bitcoin, and other networks. Deposits are automatically bridged to USDC on Polygon.

<ParamField query="wallet" type="string" required>
  The Polymarket proxy wallet address (0x...) from [Create User](/trading-api/users/create-user).
</ParamField>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://trade.predexon.com/api/bridge/deposit?wallet=0x1234567890abcdef1234567890abcdef12345678"
```

```python Python
import requests

wallet_address = "0x1234567890abcdef1234567890abcdef12345678"

response = requests.get(
    "https://trade.predexon.com/api/bridge/deposit",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"wallet": wallet_address}
)

deposit_info = response.json()

print("Deposit addresses:")
print(f"  EVM chains: {deposit_info['depositAddresses']['evm']}")
print(f"  Solana: {deposit_info['depositAddresses']['solana']}")
print(f"  Bitcoin: {deposit_info['depositAddresses']['bitcoin']}")
```

## How It Works

<Steps>
  <Step title="Get deposit addresses">
    Call this endpoint with your Polymarket wallet address.
  </Step>
  <Step title="Send funds">
    Send supported tokens to the appropriate deposit address for your source chain.
  </Step>
  <Step title="Automatic bridging">
    Funds are automatically bridged to USDC on Polygon and credited to your wallet.
  </Step>
  <Step title="Start trading">
    Check your balance and begin placing orders.
  </Step>
</Steps>

## Supported Chains

| Source Chain | Deposit Address | Supported Tokens |
|--------------|-----------------|------------------|
| Ethereum | `depositAddresses.evm` | USDC, ETH, USDT |
| Arbitrum | `depositAddresses.evm` | USDC, ETH |
| Base | `depositAddresses.evm` | USDC, ETH |
| Solana | `depositAddresses.solana` | USDC, SOL |
| Bitcoin | `depositAddresses.bitcoin` | BTC |

<Note>
Deposit addresses are cached for 15 minutes. The same address can be reused for multiple deposits.
</Note>

<Warning>
Always check `minDepositUsd` before sending. Deposits below the minimum may not be processed.
</Warning>
