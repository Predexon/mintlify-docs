---
openapi: "POST /api/users/{userId}/withdraw"
title: "Withdraw USDC"
description: "Withdraw USDC from a user's wallet to an external address"
---

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `venue` | string | Yes | Source venue: `polymarket` or `kalshi` |
| `amount` | string | Yes | USDC amount to withdraw (e.g., `"100.00"`) |
| `destinationAddress` | string | Yes | Recipient wallet address |
| `chain` | string | Yes | Destination chain: `polygon` or `solana` |

<RequestExample>
```bash cURL (Polymarket)
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "venue": "polymarket",
    "amount": "100.00",
    "destinationAddress": "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
    "chain": "polygon"
  }' \
  "https://trade.predexon.com/api/users/USER_ID/withdraw"
```

```bash cURL (Kalshi)
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
```

## Venue and Chain Combinations

| Venue | Supported Chains | Address Format |
|-------|------------------|----------------|
| `polymarket` | `polygon` | Ethereum-style (0x...) |
| `kalshi` | `solana` | Solana base58 address |

<Warning>
Ensure the destination address matches the chain. Sending to the wrong address format may result in lost funds.
</Warning>

## Before Withdrawing

<Steps>
  <Step title="Cancel open orders">
    Funds locked in orders can't be withdrawn. Use [Cancel All Orders](/trading-api/trading/cancel-all-orders) first.
  </Step>
  <Step title="Check available balance">
    Use [Get Balance](/trading-api/trading/get-balance) to verify available funds.
  </Step>
  <Step title="Withdraw">
    Call this endpoint with the withdrawal details.
  </Step>
</Steps>

<Note>
Withdrawal transactions are processed on-chain and may take a few seconds to confirm.
</Note>
