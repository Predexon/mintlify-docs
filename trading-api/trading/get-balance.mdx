---
openapi: "GET /api/users/{userId}/balance"
title: "Get Balance"
description: "Get USDC balance across trading venues"
---

<ResponseExample>

```json 200 OK
{
  "balances": [
    {
      "venue": "polymarket",
      "available": "150.50",
      "locked": "25.00",
      "total": "175.50"
    },
    {
      "venue": "kalshi",
      "available": "100.00",
      "locked": "0",
      "total": "100.00"
    }
  ]
}
```


```json 200 OK (Single Venue)
{
  "balances": [
    {
      "venue": "polymarket",
      "available": "150.50",
      "locked": "25.00",
      "total": "175.50"
    }
  ]
}
```


```json 400 Bad Request
{
  "error": "Invalid venue: 'xxx'. Must be 'polymarket' or 'kalshi'"
}
```


```json 404 Not Found
{
  "error": "User not found"
}
```

</ResponseExample>

<Note>
  Kalshi orders are atomic swaps, so they never show locked balances - funds are either spent or available.
</Note>
