---
openapi: "DELETE /api/users/{userId}/orders"
title: "Cancel All Orders"
description: "Cancel all open orders for a user"
---

<ResponseExample>
```json 200 OK
{
  "cancelled": 3
}
```

```json 200 OK (No Orders)
{
  "cancelled": 0
}
```

```json 404 Not Found
{
  "error": "User not found"
}
```
</ResponseExample>

<Note>
This only affects Polymarket orders. Kalshi orders are atomic swaps that execute immediately and cannot be cancelled.
</Note>

## Use Cases

<CardGroup cols={2}>
  <Card title="Emergency Stop" icon="hand">
    Quickly cancel all pending orders during volatile markets
  </Card>
  <Card title="Account Cleanup" icon="broom">
    Clear all orders before withdrawing funds or deleting a user
  </Card>
</CardGroup>
