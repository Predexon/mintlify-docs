---
openapi: "DELETE /api/users/{userId}/orders/{orderId}"
title: "Cancel Order"
description: "Cancel a specific open order"
---

<ResponseExample>
```text 204 No Content
(empty response body)
```

```json 400 Bad Request (Kalshi)
{
  "error": "Kalshi orders cannot be cancelled - they are atomic swaps"
}
```

```json 500 Internal Error
{
  "error": "Failed to cancel order"
}
```
</ResponseExample>

<Warning>
**Kalshi orders cannot be cancelled.** They execute atomically - either the swap completes or it fails. There's no orderbook presence to cancel.
</Warning>
