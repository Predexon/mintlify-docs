## Overview

Creates a new user with managed wallets for both supported venues:
- **Polymarket:** Gnosis Safe proxy wallet on Polygon
- **Kalshi:** Solana wallet for DFlow atomic swaps

The returned wallet addresses are used for depositing funds before trading.

<Warning>
This endpoint is blocked in Kalshi-restricted countries. See [Geo-Restrictions](/trading-api/introduction#geo-restrictions) for the full list.
</Warning>

<RequestExample>
```bash cURL
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
  "https://trade.predexon.com/api/users/create"
```

```python Python
import requests

response = requests.post(
    "https://trade.predexon.com/api/users/create",
    headers={"x-api-key": "YOUR_API_KEY"}
)
user = response.json()
print(f"User ID: {user['userId']}")
print(f"Polymarket deposit address: {user['polymarketWalletAddress']}")
print(f"Kalshi deposit address: {user['solanaWalletAddress']}")
```

```typescript TypeScript
const response = await fetch("https://trade.predexon.com/api/users/create", {
  method: "POST",
  headers: { "x-api-key": "YOUR_API_KEY" }
});
const user = await response.json();
console.log(`User ID: ${user.userId}`);
```
</RequestExample>

<ResponseField name="userId" type="string" required>
  UUID v4 identifier for the user. Store this - it's required for all subsequent operations.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the user was created.
</ResponseField>

<ResponseField name="polymarketWalletAddress" type="string" required>
  Gnosis Safe address on Polygon for Polymarket deposits. Send USDC here to fund trading.
</ResponseField>

<ResponseField name="solanaWalletAddress" type="string" required>
  Solana wallet address for Kalshi deposits. Send USDC (SPL) here for Kalshi trading.
</ResponseField>

## Next Steps

After creating a user:

<Steps>
  <Step title="Fund the wallet">
    Send USDC to the `polymarketWalletAddress` or `solanaWalletAddress` depending on which venue you want to trade on.
  </Step>
  <Step title="Check balance">
    Use [Get Balance](/trading-api/trading/get-balance) to verify funds arrived.
  </Step>
  <Step title="Place orders">
    Use [Place Order](/trading-api/trading/place-order) to start trading.
  </Step>
</Steps>

<Note>
For cross-chain deposits (e.g., depositing from Ethereum to Polygon), use the [Bridge Deposit](/trading-api/funds/deposit-info) endpoint.
</Note>
