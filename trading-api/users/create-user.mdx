---
title: "Create User"
openapi: "POST /api/users/create"
description: "Create a new user with wallets for both Polymarket and Kalshi"
---

Creates a new user with managed wallets for both supported venues. The returned wallet addresses are used for depositing funds before trading.

<Warning>
This endpoint is blocked in Kalshi-restricted countries. See [Geo-Restrictions](/trading-api/introduction#geo-restrictions) for the full list.
</Warning>

<ResponseField name="userId" type="string">
  Unique identifier for the user. Store this for subsequent API calls.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the user was created.
</ResponseField>

<ResponseField name="polymarketWalletAddress" type="string">
  Polygon wallet address for Polymarket deposits. Send USDC.e here to fund trading.
</ResponseField>

<ResponseField name="kalshiWalletAddress" type="string">
  Kalshi wallet address for deposits. Send USDC (SPL) here for Kalshi trading.
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
  "https://trade.predexon.com/api/users/create"
```

```python Python
import requests

response = requests.post(
    "https://trade.predexon.com/api/users/create",
    headers={"x-api-key": "YOUR_API_KEY"}
)
user = response.json()
print(f"User ID: {user['userId']}")
print(f"Polymarket deposit address: {user['polymarketWalletAddress']}")
print(f"Kalshi deposit address: {user['kalshiWalletAddress']}")
```

```typescript TypeScript
const response = await fetch("https://trade.predexon.com/api/users/create", {
  method: "POST",
  headers: { "x-api-key": "YOUR_API_KEY" }
});
const user = await response.json();
console.log(`User ID: ${user.userId}`);
```
</RequestExample>
