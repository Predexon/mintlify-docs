---
openapi: "POST /api/users/{userId}/redeem"
title: "Redeem Position"
description: "Redeem a resolved position for USDC"
---

## Request Body

| Parameter | Type | Polymarket | Kalshi |
|-----------|------|------------|--------|
| `venue` | string | Required (`polymarket`) | Required (`kalshi`) |
| `tokenId` | string | Required | - |
| `ticker` | string | - | Required |
| `outcome` | string | - | Required (`Yes` or `No`) |

<RequestExample>
```bash cURL (Polymarket)
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "venue": "polymarket",
    "tokenId": "71321045679252212594626385..."
  }' \
  "https://trade.predexon.com/api/users/USER_ID/redeem"
```

```bash cURL (Kalshi)
curl -X POST \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "venue": "kalshi",
    "ticker": "PRES-2024-DJT",
    "outcome": "Yes"
  }' \
  "https://trade.predexon.com/api/users/USER_ID/redeem"
```

## Redemption Flow

<Steps>
  <Step title="Check positions">
    Use [Get Positions](/trading-api/positions/get-positions) to find positions with `status: redeemable`.
  </Step>
  <Step title="Verify result">
    Only positions with `result: won` return USDC. Losing positions return 0.
  </Step>
  <Step title="Redeem">
    Call this endpoint with the position identifier.
  </Step>
  <Step title="Check balance">
    USDC is credited immediately. Verify with [Get Balance](/trading-api/trading/get-balance).
  </Step>
</Steps>

<Note>
You can redeem losing positions to clear them from your portfolio, but you'll receive 0 USDC since they're worthless.
</Note>
