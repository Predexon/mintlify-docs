## Quick Example

Fetch active markets sorted by volume:

<CodeGroup>

```bash cURL
curl "https://api.predexon.com/v1/polymarket/markets?status=open&sort=volume&limit=10"
```

```python Python
import requests

response = requests.get(
    "https://api.predexon.com/v1/polymarket/markets",
    params={"status": "open", "sort": "volume", "limit": 10}
)
markets = response.json()
```

```typescript TypeScript
const response = await fetch(
  "https://api.predexon.com/v1/polymarket/markets?status=open&sort=volume&limit=10"
);
const { markets } = await response.json();
```

</CodeGroup>
