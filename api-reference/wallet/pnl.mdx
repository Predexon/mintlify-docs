## Overview

Fetch realized profit and loss over time for a wallet address. Returns cumulative P&L at each time period based on actual closed trades and redemptions.

<Note>
This endpoint returns **realized P&L only** - from confirmed sells and redeems. For unrealized P&L, use the [Positions endpoint](/api-reference/wallet/positions).
</Note>

<ParamField path="wallet_address" type="string" required>
  Wallet address to fetch P&L for
</ParamField>

<ParamField query="granularity" type="string" required>
  Time granularity: `day`, `week`, `month`, `year`, or `all`
</ParamField>

<ParamField query="start_time" type="integer">
  Unix timestamp (seconds) for start of range
</ParamField>

<ParamField query="end_time" type="integer">
  Unix timestamp (seconds) for end of range
</ParamField>

<ParamField query="token_id" type="string">
  Optional token ID to filter P&L for a single position
</ParamField>

<ResponseField name="wallet_address" type="string">
  The queried wallet address
</ResponseField>

<ResponseField name="granularity" type="string">
  The time granularity used
</ResponseField>

<ResponseField name="start_time" type="integer">
  Start of returned data range
</ResponseField>

<ResponseField name="end_time" type="integer">
  End of returned data range
</ResponseField>

<ResponseField name="pnl_over_time" type="array">
  <Expandable title="P&L data point">
    <ResponseField name="timestamp" type="integer">
      Unix timestamp for this data point
    </ResponseField>
    <ResponseField name="pnl_to_date" type="number">
      Cumulative realized P&L up to this point in USD
    </ResponseField>
  </Expandable>
</ResponseField>

## Granularity Options

| Granularity | Data Points | Best For |
|-------------|-------------|----------|
| `day` | One per day | Short-term tracking |
| `week` | One per week | Medium-term analysis |
| `month` | One per month | Quarterly reviews |
| `year` | One per year | Multi-year analysis |
| `all` | Single value | Lifetime P&L |

## Understanding Realized P&L

Realized P&L is confirmed profit or loss from:

1. **Selling shares** - Difference between sell price and average entry price
2. **Redeeming winners** - $1.00 per share minus cost basis
3. **Losing positions** - $0.00 per share (full loss of cost basis)

<Note>
The P&L values are cumulative. Each data point shows total realized P&L **up to** that point in time.
</Note>
