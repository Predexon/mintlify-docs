---
title: "Wallet Positions"
openapi: "GET /v1/polymarket/wallet/positions/{wallet_address}"
description: "Get open and historical positions for a wallet address"
---

Fetch positions for a Polymarket wallet with cost basis, current value, and P&L.

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/polymarket/wallet/positions/0x1234...?sort_by=value&limit=50"
```

```python Python
import requests

response = requests.get(
    f"https://api.predexon.com/v1/polymarket/wallet/positions/{wallet}",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"sort_by": "unrealized_pnl", "order": "desc"}
)
data = response.json()
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/polymarket/wallet/positions/${walletAddress}?sort_by=value`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { positions, summary } = await response.json();
```
</RequestExample>
