---
title: "Candlesticks"
openapi: "GET /v1/polymarket/candlesticks/{condition_id}"
description: "Get historical OHLCV candlestick data for a market"
---

Fetch OHLCV candlestick data for charting. Prices normalized to YES side (NO prices converted as 1 - price).

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/polymarket/candlesticks/0x1234...?interval=60&start_time=1704067200&end_time=1704153600"
```

```python Python
import requests

response = requests.get(
    f"https://api.predexon.com/v1/polymarket/candlesticks/{condition_id}",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"interval": 60, "start_time": 1704067200, "end_time": 1704153600}
)
candlesticks = response.json()["candlesticks"]
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/polymarket/candlesticks/${conditionId}?interval=60`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { candlesticks } = await response.json();
```
</RequestExample>
