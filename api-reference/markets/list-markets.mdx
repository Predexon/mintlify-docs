Fetch Polymarket prediction markets with filtering, sorting, and pagination.

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/polymarket/markets?status=open&sort=volume&limit=10"
```

```python Python
import requests

response = requests.get(
    "https://api.predexon.com/v1/polymarket/markets",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"status": "open", "sort": "volume", "limit": 10}
)
data = response.json()
```

```typescript TypeScript
const response = await fetch(
  "https://api.predexon.com/v1/polymarket/markets?status=open&sort=volume&limit=10",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const data = await response.json();
```
</RequestExample>
