---
title: "Crypto Up/Down Markets"
openapi: "GET /v1/polymarket/crypto-updown"
description: "List crypto up/down prediction markets"
---

Bitcoin, Ethereum, Solana, and XRP up/down markets across all timeframes (15m, 1h, 4h, daily). Filter by asset, timeframe, status, and time range.

    }
  ],
  "pagination": {
    "limit": 50,

    "total": 240,

</ResponseExample>
