## Overview

Fetch historical time series of cumulative trading volume for a specific token. Useful for analyzing market activity trends over time.

<ParamField path="token_id" type="string" required>
  The token ID for the market
</ParamField>

<ParamField query="granularity" type="string" default="day">
  Time granularity: `day`, `week`, `month`, `year`, or `all`
</ParamField>

<ParamField query="start_time" type="integer">
  Unix timestamp (seconds) for start of range
</ParamField>

<ParamField query="end_time" type="integer">
  Unix timestamp (seconds) for end of range
</ParamField>

<ResponseField name="token_id" type="string">
  The requested token ID
</ResponseField>

<ResponseField name="granularity" type="string">
  The time granularity used
</ResponseField>

<ResponseField name="start_time" type="integer">
  Actual start time of returned data
</ResponseField>

<ResponseField name="end_time" type="integer">
  Actual end time of returned data
</ResponseField>

<ResponseField name="volume_over_time" type="array">
  <Expandable title="Volume data point">
    <ResponseField name="timestamp" type="integer">
      Unix timestamp for this data point
    </ResponseField>
    <ResponseField name="volume" type="number">
      Cumulative volume up to this point in USD
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.predexon.com/v1/polymarket/markets/21742633.../volume?granularity=day"
```

```python Python
import requests

response = requests.get(
    f"https://api.predexon.com/v1/polymarket/markets/{token_id}/volume",
    params={
        "granularity": "day",
        "start_time": 1704067200,
        "end_time": 1706745600
    }
)
volume_data = response.json()["volume_over_time"]

for point in volume_data:
    print(f"{point['timestamp']}: ${point['volume']:,.2f}")
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/polymarket/markets/${tokenId}/volume?granularity=day`
);
const { volume_over_time } = await response.json();
```

## Granularity Options

| Granularity | Description | Use Case |
|-------------|-------------|----------|
| `day` | Daily data points | Short-term analysis |
| `week` | Weekly aggregation | Medium-term trends |
| `month` | Monthly aggregation | Long-term analysis |
| `year` | Yearly aggregation | Multi-year comparison |
| `all` | Single lifetime value | Total market volume |
