## Overview

Fetch historical orderbook snapshots for a specific token. Returns bids and asks at each point in time, useful for analyzing market depth and liquidity.

<Warning>
Orderbook data history starts from **January 1st, 2026**. Earlier data is not available.
</Warning>

<ParamField query="token_id" type="string" required>
  The token ID (asset) for the Polymarket market
</ParamField>

<ParamField query="start_time" type="integer" required>
  Start time in Unix timestamp (**milliseconds**)
</ParamField>

<ParamField query="end_time" type="integer" required>
  End time in Unix timestamp (**milliseconds**)
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Maximum number of snapshots to return (1-200)
</ParamField>

<ParamField query="pagination_key" type="string">
  Pagination key for fetching the next page of results
</ParamField>

<ResponseField name="snapshots" type="array">
  <Expandable title="Orderbook snapshot">
    <ResponseField name="assetId" type="string">
      Token ID for this orderbook
    </ResponseField>
    <ResponseField name="market" type="string">
      Market condition ID
    </ResponseField>
    <ResponseField name="timestamp" type="integer">
      Snapshot timestamp in milliseconds
    </ResponseField>
    <ResponseField name="indexedAt" type="integer">
      When snapshot was indexed (milliseconds)
    </ResponseField>
    <ResponseField name="hash" type="string">
      Orderbook state hash
    </ResponseField>
    <ResponseField name="tickSize" type="string">
      Minimum price increment
    </ResponseField>
    <ResponseField name="bids" type="array">
      Bid levels ordered by price (highest first)
      <Expandable>
        <ResponseField name="price" type="string">Price level (0-1)</ResponseField>
        <ResponseField name="size" type="string">Size at this level</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="asks" type="array">
      Ask levels ordered by price (lowest first)
      <Expandable>
        <ResponseField name="price" type="string">Price level (0-1)</ResponseField>
        <ResponseField name="size" type="string">Size at this level</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  <Expandable>
    <ResponseField name="limit" type="integer">Requested limit</ResponseField>
    <ResponseField name="count" type="integer">Number of items returned</ResponseField>
    <ResponseField name="pagination_key" type="string">Key for next page (null if no more)</ResponseField>
    <ResponseField name="has_more" type="boolean">Whether more results are available</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl "https://api.predexon.com/v1/polymarket/orderbooks?token_id=21742633...&start_time=1704067200000&end_time=1704153600000&limit=50"
```

```python Python
import requests

# Note: timestamps are in MILLISECONDS
response = requests.get(
    "https://api.predexon.com/v1/polymarket/orderbooks",
    params={
        "token_id": token_id,
        "start_time": 1704067200000,  # ms
        "end_time": 1704153600000,    # ms
        "limit": 50
    }
)
snapshots = response.json()["snapshots"]

# Paginate through all results
pagination_key = response.json()["pagination"]["pagination_key"]
while pagination_key:
    response = requests.get(
        "https://api.predexon.com/v1/polymarket/orderbooks",
        params={
            "token_id": token_id,
            "start_time": 1704067200000,
            "end_time": 1704153600000,
            "pagination_key": pagination_key
        }
    )
    snapshots.extend(response.json()["snapshots"])
    pagination_key = response.json()["pagination"]["pagination_key"]
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/polymarket/orderbooks?token_id=${tokenId}&start_time=1704067200000&end_time=1704153600000`
);
const { snapshots, pagination } = await response.json();
```
