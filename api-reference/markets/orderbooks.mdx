---
title: "Orderbook History"
openapi: "GET /v1/polymarket/orderbooks"
description: "Get historical orderbook snapshots for a token"
---

Fetch historical orderbook snapshots for a token. Data starts from **January 1st, 2026**.

<Warning>Timestamps are in **milliseconds** for this endpoint.</Warning>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/polymarket/orderbooks?token_id=21742633...&start_time=1704067200000&end_time=1704153600000"
```

```python Python
import requests

response = requests.get(
    "https://api.predexon.com/v1/polymarket/orderbooks",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={
        "token_id": token_id,
        "start_time": 1704067200000,  # milliseconds
        "end_time": 1704153600000,
        "limit": 50
    }
)
snapshots = response.json()["snapshots"]
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/polymarket/orderbooks?token_id=${tokenId}&start_time=1704067200000&end_time=1704153600000`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { snapshots, pagination } = await response.json();
```
</RequestExample>
