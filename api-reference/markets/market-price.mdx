---
title: "Market Price"
openapi: "GET /v1/polymarket/market-price/{token_id}"
description: "Get the current or historical price for a specific token"
---

Fetch current or historical price for a token. Uses weighted average of recent trades when stable, or last trade price when volatile.

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/polymarket/market-price/21742633...?at_time=1704067200"
```

```python Python
import requests

response = requests.get(
    f"https://api.predexon.com/v1/polymarket/market-price/{token_id}",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"at_time": 1704067200}  # optional
)
price = response.json()["price"]
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/polymarket/market-price/${tokenId}`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { price } = await response.json();
```
</RequestExample>
