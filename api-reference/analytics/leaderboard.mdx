---
title: "Global Leaderboard"
openapi: "GET /v1/polymarket/leaderboard"
description: "Get the global leaderboard of smart wallets"
---

Ranked list of wallets by trading performance. Supports filtering by trading style (WHALE, MARKET_MAKER, ACTIVE_TRADER, etc.), PnL range, minimum volume, win rate, and more. Uses PostgreSQL for all-time data, ClickHouse for rolling windows.
