title: "Trades"
openapi: "GET /v1/dflow/trades"

---

Fetch trade history for a wallet on Dflow with market metadata.

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/dflow/trades?wallet=YOUR_WALLET_ADDRESS&limit=50"
```

```python Python
import requests

response = requests.get(
    "https://api.predexon.com/v1/dflow/trades",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"wallet": "YOUR_WALLET_ADDRESS", "limit": 50}
)
trades = response.json()["trades"]
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/dflow/trades?wallet=${walletAddress}&limit=50`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { trades } = await response.json();
```
</RequestExample>
