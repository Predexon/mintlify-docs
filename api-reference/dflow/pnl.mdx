## Overview

Fetch realized profit and loss (PnL) history for a specific wallet address on Dflow. Returns cumulative realized PnL over time with data points for every interval in the specified granularity.

<ParamField path="wallet_address" type="string" required>
  Wallet address to fetch PnL for
</ParamField>

<ParamField query="granularity" type="string" required>
  Time granularity for PnL data: `day`, `week`, `month`, `year`, or `all`
</ParamField>

<ParamField query="start_time" type="integer">
  Unix timestamp (seconds) for start of range
</ParamField>

<ParamField query="end_time" type="integer">
  Unix timestamp (seconds) for end of range
</ParamField>

<ResponseField name="wallet_address" type="string">
  The wallet address
</ResponseField>

<ResponseField name="granularity" type="string">
  The requested granularity
</ResponseField>

<ResponseField name="start_time" type="integer">
  Start of the time range (Unix timestamp)
</ResponseField>

<ResponseField name="end_time" type="integer">
  End of the time range (Unix timestamp)
</ResponseField>

<ResponseField name="pnl_over_time" type="array">
  <Expandable title="PnL data point">
    <ResponseField name="timestamp" type="integer">Unix timestamp in seconds</ResponseField>
    <ResponseField name="pnl_to_date" type="number">Cumulative realized PnL in USD up to this point</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/dflow/wallet/pnl/YOUR_WALLET_ADDRESS?granularity=day"
```

```python Python
import requests

response = requests.get(
    f"https://api.predexon.com/v1/dflow/wallet/pnl/{wallet_address}",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"granularity": "day"}
)

data = response.json()
for point in data["pnl_over_time"]:
    print(f"{point['timestamp']}: ${point['pnl_to_date']:.2f}")
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/dflow/wallet/pnl/${walletAddress}?granularity=day`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { pnl_over_time } = await response.json();

pnl_over_time.forEach(point => {
  console.log(`${point.timestamp}: $${point.pnl_to_date}`);
});
```
