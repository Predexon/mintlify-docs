title: "Wallet PnL"
openapi: "GET /v1/dflow/wallet/pnl/{wallet_address}"
description: "Fetch realized profit and loss history for a Dflow wallet"
---

Fetch cumulative realized P&L over time for a Dflow wallet.

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/dflow/wallet/pnl/YOUR_WALLET_ADDRESS?granularity=day"
```

```python Python
import requests

response = requests.get(
    f"https://api.predexon.com/v1/dflow/wallet/pnl/{wallet_address}",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"granularity": "day"}
)
pnl_data = response.json()["pnl_over_time"]
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/dflow/wallet/pnl/${walletAddress}?granularity=day`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { pnl_over_time } = await response.json();
```
</RequestExample>
