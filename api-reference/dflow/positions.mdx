## Overview

Fetch current positions for a wallet address on Dflow. Returns all positions with cost basis, realized PnL, and fees paid.

- **Open positions**: `qty > 0` (still holding tokens)
- **Closed positions**: `qty = 0` but has realized PnL history

Enriched with market metadata (title, category) from the database.

<ParamField path="wallet_address" type="string" required>
  Wallet address to fetch positions for
</ParamField>

<ParamField query="status" type="string">
  Filter by position status: `open`, `closed`, or `all` (default)
</ParamField>

<ParamField query="outcome_mint" type="string">
  Filter by specific outcome mint address
</ParamField>

<ParamField query="market_ticker" type="string">
  Filter by market ticker (case-insensitive)
</ParamField>

<ParamField query="sort_by" type="string" default="quantity">
  Field to sort by: `quantity`, `cost_basis`, `realized_pnl`, `fees_paid`
</ParamField>

<ParamField query="order" type="string" default="desc">
  Sort order: `asc` or `desc`
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Maximum number of positions to return (1-500)
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of positions to skip for pagination
</ParamField>

<ResponseField name="wallet_address" type="string">
  The wallet address
</ResponseField>

<ResponseField name="positions" type="array">

    <ResponseField name="category" type="string">Market category</ResponseField>

    <ResponseField name="fees_paid" type="number">Total fees paid in USD</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="summary" type="object">
  <Expandable>
    <ResponseField name="total_positions" type="integer">Total number of positions</ResponseField>
    <ResponseField name="total_cost_basis" type="number">Sum of all cost bases</ResponseField>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/dflow/wallet/positions/YOUR_WALLET_ADDRESS?status=open"
```

```python Python
import requests

response = requests.get(
    f"https://api.predexon.com/v1/dflow/wallet/positions/{wallet_address}",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"status": "open"}
)

data = response.json()
for pos in data["positions"]:
    print(f"{pos['title']}: {pos['qty']} shares, PnL: ${pos['realized_pnl']:.2f}")
```

```typescript TypeScript
const response = await fetch(
  `https://api.predexon.com/v1/dflow/wallet/positions/${walletAddress}?status=open`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { positions, summary } = await response.json();

positions.forEach(pos => {
  console.log(`${pos.title}: ${pos.qty} shares, PnL: $${pos.realized_pnl}`);
});
```
