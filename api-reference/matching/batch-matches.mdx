---
title: "Batch Matching Markets"
openapi: "GET /v2/matching-markets/batch"
description: "Find matching markets for multiple identifiers at once"
---

Batch version of the matching endpoint. Provide exactly one of: `polymarket_condition_ids`, `polymarket_market_slugs`, `polymarket_market_ids`, or `kalshi_market_tickers`. Returns a dictionary mapping each input identifier to its matches.
