## Overview

Find matching markets across prediction market platforms. Given a Polymarket identifier (condition ID, market slug, or market ID) OR a Kalshi market ticker, discover equivalent or related markets on the other platform.

<Warning>
Provide **exactly one** identifier parameter, not multiple.
</Warning>

<Warning>
**LLM-Based Matching**: Market matching is powered by large language models (LLMs). There is a small percentage of markets where hallucinations may occur, resulting in invalid or incorrect matches. Always verify matches before making trading decisions.
</Warning>

<RequestExample>
```bash cURL
# Using Polymarket condition ID
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/matching-markets?polymarket_condition_id=0x6f14e52f..."

# Using Polymarket market slug
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/matching-markets?polymarket_market_slug=will-chappell-roan-perform-at-the-2026-grammy-awards"

# Using Kalshi market ticker
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/matching-markets?kalshi_market_ticker=KXPERFORMGRAMMYS-26-CHA"

# Filter by match type (exact, related, or all)
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/matching-markets?kalshi_market_ticker=KXPERFORMGRAMMYS-26-CHA&match_type=exact"
```

```python Python
import requests

# Find Polymarket equivalents for a Kalshi market
response = requests.get(
    "https://api.predexon.com/v1/matching-markets",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={
        "kalshi_market_ticker": "KXPERFORMGRAMMYS-26-CHA",
        "match_type": "all"  # exact, related, or all (default)
    }
)

data = response.json()
print(f"Query: {data['query']['title']}")
print(f"Found {data['summary']['total_matches']} matches")
print(f"  Exact: {data['summary']['exact_matches']}")
print(f"  Related: {data['summary']['related_matches']}")

for match in data["matches"]["POLYMARKET"]["exact"]:
    print(f"\nExact match: {match['title']}")
    print(f"  Condition ID: {match['condition_id']}")
    print(f"  Slug: {match['market_slug']}")
    print(f"  Explanation: {match['explanation']}")
    print(f"  Similarity: {match['similarity']}")
```

```typescript TypeScript
const response = await fetch(
  "https://api.predexon.com/v1/matching-markets?kalshi_market_ticker=KXPERFORMGRAMMYS-26-CHA&match_type=all",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { query, matches, summary } = await response.json();

console.log(`Query: ${query.title}`);
console.log(`Found ${summary.total_matches} matches`);

matches.POLYMARKET.exact.forEach(match => {
  console.log(`Exact: ${match.title} (${match.market_slug})`);
  console.log(`  Condition ID: ${match.condition_id}`);
  console.log(`  Explanation: ${match.explanation}`);
  console.log(`  Similarity: ${match.similarity}`);
});
```

## Match Types

<CardGroup cols={2}>
  <Card title="Exact Matches" icon="bullseye">
    Same underlying event with equivalent or near-equivalent resolution criteria. Similarity score >= 95.
  </Card>
  <Card title="Related Matches" icon="link">
    Similar or correlated events (same entity different outcome, or same event different candidate). Similarity score between 80 and 95.
  </Card>
</CardGroup>

## Status Values

| Status | Description |
|--------|-------------|
| `active` | Market is open for trading |
| `closed` | Market is closed/resolved |

<Info>
The `similarity` field may be `null` for matches created before January 4th, 2025.
</Info>
