## Overview

Find matching markets across prediction market platforms. Given a Polymarket market slug OR a Kalshi market ticker, discover equivalent or related markets on the other platform.

<ParamField query="polymarket_market_slug" type="string">
  Polymarket market slug to find matches for
</ParamField>

<ParamField query="kalshi_market_ticker" type="string">
  Kalshi market ticker to find matches for
</ParamField>

<Warning>
Provide **exactly one** of `polymarket_market_slug` or `kalshi_market_ticker`, not both.
</Warning>

<ResponseField name="query" type="object">
  Information about the queried market
  <Expandable>
    <ResponseField name="platform" type="string">`POLYMARKET` or `KALSHI`</ResponseField>
    <ResponseField name="market_slug" type="string">Market slug (Polymarket)</ResponseField>
    <ResponseField name="market_ticker" type="string">Market ticker (Kalshi)</ResponseField>
    <ResponseField name="title" type="string">Market title</ResponseField>
    <ResponseField name="status" type="string">Market status</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="matches" type="object">
  Matches grouped by platform
  <Expandable title="Platform matches">
    <ResponseField name="exact" type="array">
      Markets representing the same underlying event
      <Expandable>
        <ResponseField name="market_slug" type="string">Market slug (Polymarket)</ResponseField>
        <ResponseField name="market_ticker" type="string">Market ticker (Kalshi)</ResponseField>
        <ResponseField name="title" type="string">Market title</ResponseField>
        <ResponseField name="status" type="string">Market status</ResponseField>
        <ResponseField name="explanation" type="string">Why this is a match</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="related" type="array">
      Markets for correlated or similar events
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="summary" type="object">
  <Expandable>
    <ResponseField name="total_matches" type="integer">Total number of matches found</ResponseField>
    <ResponseField name="exact_matches" type="integer">Number of exact matches</ResponseField>
    <ResponseField name="related_matches" type="integer">Number of related matches</ResponseField>
    <ResponseField name="platforms" type="array">List of platforms with matches</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
# From Polymarket to Kalshi
curl "https://api.predexon.com/v1/matching-markets?polymarket_market_slug=will-trump-win-2024"

# From Kalshi to Polymarket
curl "https://api.predexon.com/v1/matching-markets?kalshi_market_ticker=PRES-2024-DJT"
```

```python Python
import requests

# Find Kalshi equivalents for a Polymarket market
response = requests.get(
    "https://api.predexon.com/v1/matching-markets",
    params={"polymarket_market_slug": "will-trump-win-2024"}
)

data = response.json()
print(f"Found {data['summary']['total_matches']} matches")
print(f"  Exact: {data['summary']['exact_matches']}")
print(f"  Related: {data['summary']['related_matches']}")

for match in data["matches"]["KALSHI"]["exact"]:
    print(f"\nExact match: {match['title']}")
    print(f"  Ticker: {match['market_ticker']}")
```

```typescript TypeScript
const response = await fetch(
  "https://api.predexon.com/v1/matching-markets?polymarket_market_slug=will-trump-win-2024"
);
const { matches, summary } = await response.json();

console.log(`Found ${summary.total_matches} matches`);
matches.KALSHI.exact.forEach(match => {
  console.log(`Exact: ${match.title} (${match.market_ticker})`);
});
```

## Match Types

  <Card title="Exact Matches" icon="bullseye">

  </Card>
  <Card title="Related Matches" icon="link">
    Similar or correlated events. Useful for hedging and research.
