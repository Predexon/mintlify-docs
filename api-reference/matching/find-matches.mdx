## Overview

Find matching markets across prediction market platforms. Given a Polymarket market slug OR a Kalshi market ticker, discover equivalent or related markets on the other platform.

<Warning>
Provide **exactly one** of `polymarket_market_slug` or `kalshi_market_ticker`, not both.
</Warning>

<RequestExample>
```bash cURL
# From Polymarket to Kalshi
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/matching-markets?polymarket_market_slug=will-the-carolina-hurricanes-win-the-metropolitan-division-123"

# From Kalshi to Polymarket
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/matching-markets?kalshi_market_ticker=KXNHLMETROPOLITAN-26-CAR"
```

```python Python
import requests

# Find Polymarket equivalents for a Kalshi market
response = requests.get(
    "https://api.predexon.com/v1/matching-markets",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"kalshi_market_ticker": "KXNHLMETROPOLITAN-26-CAR"}
)

data = response.json()
print(f"Query: {data['query']['title']}")
print(f"Found {data['summary']['total_matches']} matches")
print(f"  Exact: {data['summary']['exact_matches']}")
print(f"  Related: {data['summary']['related_matches']}")

for match in data["matches"]["POLYMARKET"]["exact"]:
    print(f"\nExact match: {match['title']}")
    print(f"  Slug: {match['market_slug']}")
    print(f"  Explanation: {match['explanation']}")
    print(f"  Similarity: {match['similarity']}")
```

```typescript TypeScript
const response = await fetch(
  "https://api.predexon.com/v1/matching-markets?kalshi_market_ticker=KXNHLMETROPOLITAN-26-CAR",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { query, matches, summary } = await response.json();

console.log(`Query: ${query.title}`);
console.log(`Found ${summary.total_matches} matches`);

matches.POLYMARKET.exact.forEach(match => {
  console.log(`Exact: ${match.title} (${match.market_slug})`);
  console.log(`  Explanation: ${match.explanation}`);
  console.log(`  Similarity: ${match.similarity}`);
});
```

## Match Types

<CardGroup cols={2}>
  <Card title="Exact Matches" icon="bullseye">
    Same underlying event with equivalent resolution criteria. Ideal for arbitrage. Similarity score >= 0.95.
  </Card>
  <Card title="Related Matches" icon="link">
    Similar or correlated events (same entity different outcome, or same event different candidate). Useful for hedging and research. Similarity score between 0.90 and 0.95.
  </Card>
</CardGroup>

<Info>
The `similarity` field may be `null` for matches created before January 4th, 2025.
</Info>
