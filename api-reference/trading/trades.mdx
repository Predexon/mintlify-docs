---
title: "Trades History"
openapi: "GET /v1/polymarket/trades"
description: "Fetch historical trade data"
---

Query historical trade data with flexible filtering by market, token, user, and time range. At least one of `market_slug`, `condition_id`, `token_id`, or `user` is required.

```json 200
      "condition_id": "0x1a2b3c4d...",
      "shares": 500.0,
      "shares_normalized": 500.0,
      "price": 0.62,
      "tx_hash": "0xabc123...",

    }
  ],
  "pagination": {
    "limit": 50,

}
```

```json 400
{
  "error": "Invalid filter combination",
  "message": "Only one of market_slug, token_id, or condition_id can be provided"
```
