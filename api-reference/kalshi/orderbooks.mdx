title: "Orderbook History"
openapi: "GET /v1/kalshi/orderbooks"
description: "Fetch historical orderbook snapshots for a Kalshi market"
---

Fetch historical orderbook snapshots for a Kalshi market. Data starts from **January 7th, 2026**.

<Warning>Timestamps are in **milliseconds**. Prices are in **cents** (1-99).</Warning>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.predexon.com/v1/kalshi/orderbooks?ticker=KXBTC-25JAN31-B100000&start_time=1706745600000&end_time=1706749200000"
```

```python Python
import requests
import time

end_time = int(time.time() * 1000)  # now in ms
start_time = end_time - (60 * 60 * 1000)  # 1 hour ago

response = requests.get(
    "https://api.predexon.com/v1/kalshi/orderbooks",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"ticker": ticker, "start_time": start_time, "end_time": end_time}
)
snapshots = response.json()["snapshots"]
```

```typescript TypeScript
const endTime = Date.now();
const startTime = endTime - 60 * 60 * 1000;

const response = await fetch(
  `https://api.predexon.com/v1/kalshi/orderbooks?ticker=KXBTC-25JAN31-B100000&start_time=${startTime}&end_time=${endTime}`,
  { headers: { "x-api-key": "YOUR_API_KEY" } }

const { snapshots, pagination } = await response.json();
```
