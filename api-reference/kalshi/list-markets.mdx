---
title: "List Markets"
openapi: "GET /v1/kalshi/markets"
description: "Query Kalshi markets with filtering, sorting, and pagination"
---

Fetch Kalshi prediction markets with filtering, sorting, and pagination. Does not include Multi Variate Event (MVE) markets like parlays. Prices returned as decimals (0-1).
