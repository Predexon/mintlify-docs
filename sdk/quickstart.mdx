## Polymarket (EVM)

<Steps>
  <Step title="Create the SDK instance">
    ```typescript
    import { PredexonSDK, createViemAdapter } from '@predexon/trade-sdk';

    // walletClient is a viem WalletClient â€” see Wallet Adapters for full setup
    const wallet = createViemAdapter(walletClient);
    const walletAddress = await wallet.account.getAddress();

    const sdk = new PredexonSDK({
      apiKey: process.env.PREDEXON_API_KEY!,
      wallet,
    });
    ```
  </Step>
  <Step title="Enable trading (one-time setup)">
    ```typescript
    await sdk.enableTrading({
      core: { venue: 'polymarket', wallet: walletAddress },
      approvals: 'broadcast', // or 'sign' if wallet supports signTransaction
    });
    ```

    <Info>
    If you omit `approvals`, the SDK auto-selects based on your wallet's capabilities. See [Wallet Adapters](/sdk/wallet-adapters) for details.
    </Info>
  </Step>
  <Step title="Place a market order">
    ```typescript
    const result = await sdk.placeOrder({
      core: {
        venue: 'polymarket',
        wallet: walletAddress,
        side: 'buy',
        amount: '10', // $10 USDC
        type: 'market',
      },
      flags: { tokenId: 'your-token-id' },
    });

    console.log('Order placed:', result.orderId, result.status);
    ```

    <Info>
    Get `tokenId` from the [markets list endpoint](/api-reference/markets/list-markets). Each market has two tokenIds (Yes/No outcomes).
    </Info>
  </Step>
</Steps>

## Kalshi (Solana)

<Steps>
  <Step title="Create the SDK with Solana adapter">
    ```typescript
    import { PredexonSDK, createSolanaAdapter } from '@predexon/trade-sdk';

    // See Wallet Adapters for full Solana signing setup
    const solanaSigner = createSolanaAdapter({
      signSolanaTransaction: async (base64Tx) => { /* sign and return base64 */ },
      signSolanaMessage: async (message) => { /* sign and return base64 */ },
    });
    const walletAddress = 'your-solana-wallet-address';

    const sdk = new PredexonSDK({
      apiKey: process.env.PREDEXON_API_KEY!,
      solanaSigner,
    });
    ```
  </Step>
  <Step title="Enable trading (wallet registration)">
    ```typescript
    await sdk.enableTrading({
      core: { venue: 'kalshi', wallet: walletAddress },
    });
    ```
  </Step>
  <Step title="Place a market order">
    ```typescript
    const result = await sdk.placeOrder({
      core: {
        venue: 'kalshi',
        wallet: walletAddress,
        side: 'buy',
        amount: '10', // $10 USDC
        type: 'market',
      },
      flags: { ticker: 'TICKER-SYMBOL', outcome: 'Yes' },
    });

    console.log('Order placed:', result.orderId, result.status);
    ```

    <Info>
    Get `ticker` from the [Kalshi markets endpoint](/api-reference/kalshi/list-markets).
    </Info>
  </Step>
</Steps>
