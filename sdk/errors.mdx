The SDK throws typed errors for different failure scenarios, allowing you to handle each case appropriately.

## Error Types

| Error Class | Code | Description |
|-------------|------|-------------|
| `SdkError` | `SDK_ERROR` | Base error class for all SDK errors |
| `NetworkError` | `NETWORK_ERROR` | HTTP/network failures, timeouts |
| `AuthError` | `AUTH_ERROR` | API key invalid or wallet authentication issues |
| `ValidationError` | `VALIDATION_ERROR` | Invalid input parameters |
| `SigningError` | `SIGNING_ERROR` | Wallet signing failures |
| `OrderRejectedError` | `ORDER_REJECTED` | Order rejected by venue (insufficient balance, liquidity, etc.) |
| `NotSupportedError` | `NOT_SUPPORTED` | Unsupported operation (e.g., limit orders on Kalshi) |

## Importing Errors

```typescript
import {
  SdkError,
  NetworkError,
  AuthError,
  ValidationError,
  SigningError,
  OrderRejectedError,
  NotSupportedError,
} from '@predexon/trade-sdk';
```

## Basic Error Handling

```typescript
try {
  await sdk.placeOrder({
    core: {
      venue: 'polymarket',
      wallet: '0x...',
      side: 'buy',
      amount: '10',
      type: 'market',
    },
    flags: { tokenId: '12345...' },
  });
} catch (error) {
  if (error instanceof ValidationError) {
    console.log('Invalid input:', error.message);
    console.log('Error code:', error.code); // 'VALIDATION_ERROR'
  } else if (error instanceof AuthError) {
    console.log('Authentication failed:', error.message);
  } else if (error instanceof NetworkError) {
    console.log('Network issue:', error.message);
  } else if (error instanceof SigningError) {
    console.log('Wallet signing failed:', error.message);
  } else if (error instanceof OrderRejectedError) {
    console.log('Order rejected:', error.message);
  } else if (error instanceof NotSupportedError) {
    console.log('Not supported:', error.message);
  }
}
```

## Error Handling by Scenario

### Validation Errors

Thrown when input parameters are invalid. Fix the request before retrying.

```typescript
try {
  await sdk.placeOrder({
    core: { venue: 'polymarket', wallet: '0x...', side: 'buy', type: 'market' },
    flags: { tokenId: '' }, // Invalid: empty tokenId
  });
} catch (error) {
  if (error instanceof ValidationError) {
    // Fix the input and retry
    console.log('Validation failed:', error.message);
  }
}
```

### Network Errors

Thrown for HTTP failures, timeouts, or connectivity issues. Safe to retry.

```typescript
try {
  await sdk.placeOrder({
    core: { venue: 'polymarket', wallet: '0x...', side: 'buy', amount: '10', type: 'market' },
    flags: { tokenId: '12345...' },
  });
} catch (error) {
  if (error instanceof NetworkError) {
    // Safe to retry with same idempotency key
    console.log('Network error, retrying...');
  }
}
```

### Auth Errors

Thrown when API key is invalid or wallet authentication fails.

```typescript
try {
  await sdk.enableTrading({ core: { venue: 'polymarket', wallet: '0x...' } });
} catch (error) {
  if (error instanceof AuthError) {
    // Check API key or wallet connection
    console.log('Auth failed:', error.message);
  }
}
```

### Signing Errors

Thrown when the wallet fails to sign a transaction or typed data.

```typescript
try {
  await sdk.placeOrder({
    core: { venue: 'polymarket', wallet: '0x...', side: 'buy', amount: '10', type: 'market' },
    flags: { tokenId: '12345...' },
  });
} catch (error) {
  if (error instanceof SigningError) {
    // User may have rejected the signature request
    console.log('Signing failed:', error.message);
  }
}
```

### Order Rejected Errors

Thrown when the venue rejects the order (e.g., insufficient balance, no liquidity).

```typescript
try {
  await sdk.placeOrder({
    core: { venue: 'polymarket', wallet: '0x...', side: 'buy', amount: '10', type: 'market' },
    flags: { tokenId: '12345...' },
  });
} catch (error) {
  if (error instanceof OrderRejectedError) {
    // Check balance, liquidity, or order parameters
    console.log('Order rejected:', error.message);
  }
}
```

## Complete Error Handler

```typescript
import {
  PredexonSDK,
  ValidationError,
  AuthError,
  NetworkError,
  SigningError,
  OrderRejectedError,
  NotSupportedError,
  PlaceOrderRequest,
} from '@predexon/trade-sdk';

async function placeOrderSafely(sdk: PredexonSDK, request: PlaceOrderRequest) {
  try {
    const result = await sdk.placeOrder(request);
    return { success: true, order: result };
  } catch (error) {
    if (error instanceof ValidationError) {
      // Invalid input - fix the request
      return { success: false, error: 'validation', message: error.message };
    }
    if (error instanceof AuthError) {
      // Auth issue - check API key or wallet
      return { success: false, error: 'auth', message: error.message };
    }
    if (error instanceof NetworkError) {
      // Network issue - retry later
      return { success: false, error: 'network', message: error.message };
    }
    if (error instanceof SigningError) {
      // Signing failed - user may have rejected
      return { success: false, error: 'signing', message: error.message };
    }
    if (error instanceof OrderRejectedError) {
      // Order rejected - check liquidity/balance
      return { success: false, error: 'rejected', message: error.message };
    }
    if (error instanceof NotSupportedError) {
      // Operation not supported
      return { success: false, error: 'not_supported', message: error.message };
    }
    throw error; // Unknown error
  }
}
```
