The SDK is written in TypeScript and exports all types for full type safety.

## Importing Types

```typescript
import type {
  // Core types
  Venue,
  OrderSide,

  // Request types
  PlaceOrderRequest,
  PrepareOrderRequest,
  EnableTradingRequest,
  RedeemRequest,

  // Response types
  PlaceOrderResponse,
  EnableTradingResponse,
  RedeemResponse,
  OpenOrder,
  Position,
  Balance,

  // Flags
  PolymarketFlags,
  KalshiFlags,

  // Wallet types
  EvmSigner,
  SolanaSigner,
  PrivySigner,
  Eip712TypedData,
  EvmTransactionRequest,

  // Config
  PredexonSdkConfig,

  // Order types
  PolymarketOrder,
  KalshiOrderParams,

  // Approval types
  ApprovalStatus,
  ApprovalTransaction,
} from '@predexon/trade-sdk';
```

## Core Types

### Venue

```typescript
type Venue = 'polymarket' | 'kalshi';
```

### OrderSide

```typescript
type OrderSide = 'buy' | 'sell';
```

## Request Types

### PlaceOrderRequest

```typescript
interface PlaceOrderRequest {
  core: {
    venue: Venue;
    wallet: string;
    side: OrderSide;
    type: 'market' | 'limit';
    amount?: string;  // USDC amount (market buy)
    size?: string;    // Share count
    price?: string;   // Price per share (limit orders)
  };
  flags: PolymarketFlags | KalshiFlags;
  idempotencyKey?: string;
}
```

### EnableTradingRequest

```typescript
interface EnableTradingRequest {
  core: {
    venue: Venue;
    wallet: string;
  };
  approvals?: 'auto' | 'manual' | 'skip'; // Polymarket only
}
```


## Module Formats

The SDK ships as dual ESM/CJS:

<CodeGroup>
```typescript ESM (recommended)
import { PredexonSDK } from '@predexon/trade-sdk';
```

```javascript CommonJS
const { PredexonSDK } = require('@predexon/trade-sdk');
```
</CodeGroup>

## Type-Safe Usage Example

```typescript
import {
  PredexonSDK,
  createEvmAdapter,
  type PlaceOrderRequest,
  type PlaceOrderResponse,
  type PolymarketFlags,
} from '@predexon/trade-sdk';

const sdk = new PredexonSDK({
  apiKey: process.env.PREDEXON_API_KEY!,
  wallet: createEvmAdapter(walletClient),
});

// Fully typed request
const request: PlaceOrderRequest = {
  core: {
    venue: 'polymarket',
    wallet: '0x...',
    side: 'buy',
    amount: '10',
    type: 'market',
  },
  flags: {
    tokenId: '12345...',
  } as PolymarketFlags,
};

// Fully typed response
const response: PlaceOrderResponse = await sdk.placeOrder(request);
console.log(response.orderId, response.status);
```
